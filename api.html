Authentication

 

HighPass API uses token-based authentication. All requests require authentication token to be set in headers: 

curl "https://highpass.aero:444/api/v1/v1.0?{request paramenters}"
  -X GET
  -H "Authorization: Bearer {access token}"
  -H "Accept-Language: en-US"

API methods

Get airports

Returns available airports by query.

 

API endpoint

GET /api/v1/airports?query={airport query}&pageSize={page size}&page={page}

 

Request parameters
query Airport search string, min length is 3 characters.
Search is performed by the following: Airport name, Airport IATA code, Country, City
pageSize Amount of elements to be returned on a single response page
page Current response page 
curl "https://highpass.aero:444/api/v1/airports?query=hrk&pageSize=10&page=10"
  -X GET
  -H "Authorization Bearer {access_token}"

 

Response parameters
query Airport search string sent in the request
page Response page sent in the request
pageSize Amount of airports to be returned sent in the request
total Amount of airports found by the query parameter regardless parameters page and pageSize
airports A list of airports found - see below
Airports parameters
id Airport ID
name Airport name
city A city airport relates to
country
iataCode Three-character IATA code

	"page" : 1,
	"pageSize" : 5,

			"name" : "Kharkov, Osnova International Airport, Ukraine (HRK)",
			"city" : "Kharkov",
			"country" : "Ukraine", 
			"iataCode" : "HRK"
		}
	]

Get service price

Returns a total service price for selected amount of people.

 

API endpoint

GET /api/v1/services/{service_id}/price?adultCount={adults}&childCount={children}&infantCount={infants}

 

Request parameters
service_id Service id
adultCount Amount of adults in the order
childCount Amount of children in the order
infantCount Amount of infants in the count 
curl "https://highpass.aero:444/api/v1/services/d8e2b06d-b8c1-89b0-3004-4793fdc08cad/price?adultCount=2&childCount=1&infantCount=0"
  -X GET
  -H "Authorization Bearer {access_token}"
Response parameters
price Total price for all people in the order, EUR 
{
	"price" : 32.55
}

Create order

Creates an order with specified parameters and returns and order id.

 

API endpoint

POST /api/v1/orders

 

Request parameters
serviceId Service id
flightNumber Customer(s) flight number to verify a service time
serviceDate Order date
adultCount Amount of adults in the order
childCount Amount of children in the order
infantCount Amount of infants in the count
comment Additional information specified by the customer
firstName Customer's first name
lastName Customer's last name
phone Customer's contact phone
email Customer's contact email
otherPassengersContactDetails A free-text parameter, required if a total amount of people in the order is grater then one. 
curl "https://highpass.aero:444/api/v1/orders"
    -X POST
    -H "Authorization: Bearer {access_token}"
    -d 'serviceId=d8e2b06d-b8c1-89b0-3004-4793fdc08cad
		&flightNumber=PC 751
		&serviceDate=02.04.2018 13:35:00
		&adultCount=2
		&childCount=1
		&infantCount=0
		&comment=we're travelling with a dog
		&firstName=John
		&lastName=Doe
		&phone=+1 555-55-55
		&email=john.doe@email.com
		&otherPassengersContactDetails=Marissa Doe, Ben Doe'

 

Response parameters
price Total price for all people in the order, EUR 
{
	"orderId" : "d8e2b06d-b8c1-89b0-3004-4793fdc08cad"
}

Complete order

Finishes an order booking and returns a booking code.

 

API endpoint

GET /api/v1/orders/{order_id}/completionDetails

 

Request parameters
order_id Order id recieved with Create order method 
curl "https://highpass.aero:444/api/v1/orders/d8e2b06d-b8c1-89b0-3004-4793fdc08cad/completionDetails"
    -X GET
    -H "Authorization: Bearer {access_token}"

 

Response parameters
orderDetails Order booking information - see below
orderDetails parameters
competionCode A code a customer must provide to a service provide to receive a booked service
orderNumber

		"competionCode" : "XDLBFN",
