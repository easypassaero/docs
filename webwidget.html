Web Widget

Last updated: March 20th, 2017

API Key

HighPass uses API keys to allow access to the APIs. You can create API keys through your HighPass Agent account.

Widget configuration

Basic setup

Paste the following line into your page:

Basic setup example
<script id="highpass-web-widget" src="https://highpass.aero/widget/Init?apiKey=[Your API Key]" type="text/javascript"></script>

Extended setup

Or add additinal parameters to customize widget appearance and behavior:

Extended setup example

<script id="highpass-web-widget" src="https://highpass.aero/widget/Init?apiKey=[Your API Key][URL encoded parameters]" type="text/javascript"></script>
